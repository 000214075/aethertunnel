## 🎨 核心设计原则

### 1. 视觉一致性

**主色调**：
- 主色：`#6366f1` (蓝紫色渐变）
- 辅助色：`#ec4899` (粉色)
- 成功色：`#10b981` (绿色)
- 警告色：`#f59e0b` (橙色)
- 危险色：`#ef4444` (红色)

**渐变色方案**：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 2. 卡片式设计

所有内容块都采用卡片式设计：
- 白色背景
- 圆角 16px
- 阴影效果
- 悬浮动画

### 3. 动画效果

**过渡动画**：
```css
transition: all 0.3s ease;
```

**悬停效果**：
```css
transform: translateY(-2px);
box-shadow: 0 20px 25px rgba(0, 0, 0, 0.15);
```

**加载动画**：
- 旋转的加载圈
- 脉冲动画（语言选择）
