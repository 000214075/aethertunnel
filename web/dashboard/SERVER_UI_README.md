- 网格背景

- 实时更新
- 流畅动画

**实现方式**：
```javascript
// 使用 Canvas 2D 绘制
const ctx = canvas.getContext('2d');

// 绘制渐变曲线
const gradient = ctx.createLinearGradient(0, 0, width, height);
gradient.addColorStop(0, 'rgba(102, 126, 234, 0.8)');
gradient.addColorStop(1, 'rgba(118, 75, 162, 0.8)');
ctx.strokeStyle = gradient;
ctx.lineWidth = 3;
```
