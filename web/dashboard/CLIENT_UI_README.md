**配置项**：

**功能**：

- 日志级别过滤（全部、信息、警告、错误）

## 🎯 核心功能

### 1. 代理管理

**添加代理**：
- 模态对话框
- 表单验证
- 多种代理类型：
  - TCP（SSH、数据库等）
  - UDP（DNS、游戏等）
  - HTTP（Web 网站）
  - HTTPS（加密网站）
  - STCP（安全 TCP）
  - XTCP（P2P 直连）

**编辑代理**：
- 修改配置
- 实时保存
- 配置验证

**删除代理**：
- 确认对话框
- 删除确认

**批量操作**：
- 全部启动
- 全部停止
- 全部重启

### 2. 连接管理

**连接状态**：
- 已连接（绿色）
- 未连接（红色）
- 连接中（橙色，旋转）

**连接信息**：
- 服务器地址
- 连接时间
- 活跃隧道数
- 延迟信息

**连接操作**：
- 连接到服务器
- 断开连接

### 3. 客户端配置

**连接配置**：
- 服务器地址（IP:端口）
- 认证令牌（密码）
- 重连间隔
- 最大重连次数

**TLS 配置**：
- 启用 TLS
- 跳过证书验证

### 4. 流量统计

**实时统计**：
- 活跃隧道数
- 总流量使用
- 平均延迟
- 连接率

**图表展示**：
- 流量趋势图
- 实时更新
- 多时间范围

### 5. 日志查看

**日志级别**：
- INFO（蓝色）
- WARN（橙色）
- ERROR（红色）

**日志操作**：
- 级别过滤
- 自动滚动
- 清空日志
